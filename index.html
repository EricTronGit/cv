Tarkov Challenge

Tarkov challenge is a project i want to make in React. It's simple, Escape from Tarkov is a hardcore first person shooter. But at some point, the game can be boring, because you're not having anything to do. So with this application you can select many option to create a challenge to achieve. For exemple on Twitch, a viewer ask for Tarkov Challenge, the streamer go on this application, click on ROLL and he's giving a challenge. The streamer need to achieve this challenge in the game. It's for making live streaming more entertainment.

You can see the app here : https://tarkovapp.firebaseapp.com/
